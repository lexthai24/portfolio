# Portfolio — Ekkawit P.

Personal portfolio site. Full-stack developer, Next.js / TypeScript / Node.js.

## Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion

## Development

```bash
npm install
npm run dev
```

Open http://localhost:3000.

## Editing content

All copy lives in [`lib/data.ts`](lib/data.ts) — profile, skills, projects, and
experience. Client project details are anonymized to respect NDAs.

## Build

```bash
npm run build
npm run start
```

## Deploy

Deploys to Vercel out of the box — import the repo and ship.
